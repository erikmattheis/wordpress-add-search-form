Adds a search form to most WordPress Themes that don\'t have one.

== Description ==
This plugin adds a search form to most WordPress Themes that don\'t have one. It will insert a form containing a search field within the element with the ID attribute you specify.

== Installation ==
No special requirements. After the plugin is installed find the "Add Search Form Settings" link in the admin menu. Indicate the id or class of the HTML element you wish choose as the search form\'s parent element and wether you with the search form to be the first or last child. NOTE: The plugin does not create a search results page, you are on your own for that!
